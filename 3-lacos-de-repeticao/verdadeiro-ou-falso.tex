\question[10]

Use (\textbf{V}) para verdadeiro e (\textbf{F}) para falso nas seguintes sentenças.

\begin{parts}
	
	\part \tf[F] Os comandos \emph{for}, \emph{while} e \emph{switch} são exempos de comandos de repetição.

	\part \tf[F] É necessário que o contador do laço seja do tipo \emph{int}. Não pode ser \emph{float}.

	\part \tf[F] O comando \emph{do-while} é usado para garantir que o código será executado pelo menos duas vezes.
	
\end{parts}
